# Product view card

Minimal and responsive Product view card built with HTML and CSS. This project focuses on responsive design, accessibility, clean UI structure, and pixel-perfect.

## Demo


https://product-view-card.cristiancode.com


## Technologies Used

- HTML5
- CSS3
- SASS
- Flexbox
- CSS Grid
- Responsive Design

## Installation

Clone the repository:

```bash
git clone https://github.com/cristiankode/product-view-card.git
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
